    Présentation du Site

  • Notre site a été créé dans le cadre de l'ISN. Pour comprendre ce qu'est l'ISN, je vous invite à lire les paragraphes se trouvant sur le côté. Lors de notre année de terminale, nous avons du réaliser un projet informatique. Après plusieurs essais dans différents domaines (création vidéo par exemple), nous avons finalement choisis de créer un site web dynamique qui pourrait aider les usagers dans leur orientation professionelle. Pour cela, nous avons décidé de créer chacun nos propres pages puis de les regrouper sur ce site.

    Qu'est-ce que l'ISN ?
    L'ISN (Informatique et Sciences du Numérique) est une option que nous avons prise pour notre année de terminale. Dans cette option, nous apprenons les différents languages de programmation c'est à dire le java, le C++, le Html / CSS... Durant cette année nous avons à réaliser un projet informatique de notre choix qui sera évalué à la fin de l'année scolaire.
